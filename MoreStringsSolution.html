String Methods

Problem: Given the string "The quick brown fox jumps over the lazy dog.", replace "lazy dog" with "energetic puppy" using the replace() method and print the new string.

original_string = "The quick brown fox jumps over the lazy dog."
new_string = original_string.replace(CODE, CODE)
CODE
# Solution:
original_string = "The quick brown fox jumps over the lazy dog."
new_string = original_string.replace("lazy dog", "energetic puppy")
print(new_string)
The quick brown fox jumps over the energetic puppy.

Problem: Given the string "book,pen,paper,notebook", split it into a list of substrings using the split() method and print the list.

original_string = "book-pen-paper-notebook"
item_list = original_string.split(CODE)
CODE
# Solution:
original_string = "book-pen-paper-notebook"
item_list = original_string.split('-')
print(item_list)
['book', 'pen', 'paper', 'notebook']

word_list = ['Python', 'is', 'awesome']
CODE
# Solution:
word_list = ['Python', 'is', 'awesome']
result_string = 'xoxoxo'.join(word_list)
