import textwrap
from langchain_core.prompts import ChatPromptTemplate

def wrap_text(text, max_width=80):
    """
    Wraps the text to the specified max_width, preserving line breaks and formatting.
    """
    text = text.lstrip()
    lines = text.splitlines()  # Split the text into lines
    wrapped_lines = []
    for line in lines:
        if line.strip():  # Skip empty lines
            wrapped_line = textwrap.fill(line, max_width, initial_indent='', subsequent_indent='')
            wrapped_lines.extend(wrapped_line.splitlines())  # Preserve line breaks
        else:
            wrapped_lines.append('')  # Keep empty lines
    return '\n'.join(wrapped_lines)

def print_prompt_and_response(prompt, response):
    print("Prompt: ")
    print(wrap_text(prompt))
    print("")
    print("Response: ")
    print(response)

def print_messages_and_response(messages, response):
    prompt = ChatPromptTemplate(messages=messages)
    print_prompt_and_response(prompt.format(), response)

def get_completion(prompt, temperature=0.0):
    response = llm.invoke(prompt, temperature=temperature)
    wrapped_response = wrap_text(response.content)
    return wrapped_response

def get_completion_messages(messages, temperature=0.0):
    response=llm.invoke(messages, temperature=temperature)
    wrapped_response = wrap_text(response.content)
    return wrapped_response

A black hole is a region in spacetime exhibiting gravitational acceleration so
strong that nothing—no particles or even electromagnetic radiation such as
light—can escape from it. It arises from the gravitational collapse of a massive
star, which causes an infinitely dense point called a singularity. Black holes
are characterized by their event horizon, or the boundary beyond which the
escape velocity exceeds the speed of light, effectively forming an opaque
barrier preventing communication with regions beyond the horizon.

How many key strokes to type the numbers 1 to 500?

So this is a problem where LLM has to realize how many digits are in the numbers as it tries to figure this out.

  • 1 through 9 are just one digit, so one keystroke for them
  • 10 through 99 are two digits, so two keystrokes for them
  • 100 through 500 are three digits, so three keystrokes for them

There are 9 different numbers with 1 digit, 90 different numbers with 2 digits, and 401 different numbers with 3 digits so:

\[ 9(1) + 90(2) + 401(3) = 1392 \]

Lets ask the LLM first without any role.

  • Ask the LLM how many key strokes are needed to type the numbers from 1 to 500?
prompt = """
CODE
"""

print(response)
# Solution:
prompt="""
How many key strokes are needed to type the numbers from 1 to 500?
"""

response=get_completion(prompt)
print(response)
To calculate the number of keystrokes needed to type the numbers from 1 to 500,
we need to consider that each number from 1 to 9 requires one keystroke, while
numbers from 10 to 500 require two keystrokes (one for the tens digit and one
for the ones digit).

There are 49 numbers from 1 to 9, so typing these numbers requires 49
keystrokes.

For the numbers from 10 to 500, there are 49 (500 - 10 + 1) numbers in this
range. Each number has two digits, so there are 2 * 49 = 98 keystrokes for these
numbers.

Adding the keystrokes for the numbers from 1 to 9 and from 10 to 500, we get 49
+ 98 = 147 keystrokes.

Therefore, it would take approximately 147 keystrokes to type the numbers from 1
to 500.

To determine the number of keystrokes needed to type the numbers from 1 to 500,
we need to consider that:

1. Each number from 1 to 9 only requires one keystroke (the number itself).
2. Numbers from 10 to 99 require two keystrokes (the first digit and the second
digit).
3. Numbers from 100 to 500 require three keystrokes (the first digit, the second
digit, and the letter "h" for hundreds).

Now, let's calculate the number of keystrokes for each range:

1. For numbers 1 to 9, there are 9 numbers, so it requires 9 keystrokes.
2. For numbers 10 to 99, there are 90 numbers (9 numbers per decade times 10
decades), so it requires 90 x 2 = 180 keystrokes.
3. For numbers 100 to 500, there are 401 numbers (5 decades times 100 plus 1),
so it requires 401 x 3 = 1203 keystrokes.

Adding up the keystrokes for each range, we get:

9 + 180 + 1203 = 1392 keystrokes.

So, it would take 1392 keystrokes to type the numbers from 1 to 500.
